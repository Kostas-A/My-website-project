Έχουμε σχεδιάσει αρκετά μικρά προγράμματα ερασιτεχνικά στο παρελθόν και έχοντας την
απαραίτητη πλέον εμπειρία σκεφτήκαμε να ιδρύσουμε αυτή την εταιρία ώστε να μπορέσουμε
να αξιοποιήσουμε ακόμα περισσότερο τις γνώσεις μας για τον προγραμματισμό στις γλώσσες
html, css και sql, καθώς και να αποκτήσουμε περισσότερη εμπειρία με αυτόν.
Παρακάτω σας αναφέρουμε ένα project που πιστεύουμε ότι είναι το σημαντικότερο που
έχουμε υλοποιήσει ως τώρα:

Σχεδιάσαμε ένα πρόγραμμα στο οποίο, αφού οι χρήστες καταχωρούσαν τα δεδομένα τους σε
μια ιστοσελίδα, τα δεδομένα αυτά στέλλονταν και αποθηκεύονταν σε κάποιους server ώστε
να γίνει η αξιολόγηση τους από στελέχη της εταιρίας. Μετά την αξιολόγηση, τα δεδομένα
αυτά χωρίζονταν σε τρεις κατηγορίες, είτε ήταν έγκυρα, είτε ήταν άκυρα, είτε ανακριβή.

Ένα άλλο project που θα θέλαμε να σχεδιάσουμε μελλοντικά, βασίζεται στο προηγούμενο
πρόγραμμα και συλλέγει τα δεδομένα των χρηστών, με σκοπό να ερευνήσει τις συμπεριφορές
των ατόμων με βάση τις επιλογές που έκανα και γιατί τις έκαναν. Για την υλοποίηση αυτού
του project θα συνεργαστούμε και με άλλους προγραμματιστές, που ειδικεύονται στην τεχνητή νοημοσύνη.
